# Jekyll with a tiny footprint

[Demo](http://rpiai.com/jekyll-minimal/)

## Footprint statistics

| Page        | Transfered           |
| ------------- |:-------------:|
| Typical page   | ~3.1 KB |
| Main CSS + JS   | ~2.2 KB |
| Analytics (optional)   | ~11.2 KB |
| Disqus (optional)   | ~300 KB |

## Features

- Very small footprint (CSS+JS < 10 KB)
- Fixed navbar
- Good SEO
- Tag page
- Tags underneath each post title
